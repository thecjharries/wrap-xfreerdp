# `wrap-xfreerdp`

## Installation

    git clone <repo>
    npm install

### Configs

#### `.editorconfig`

The excellent [Editor Config](http://editorconfig.org/) makes editor settings as consistent as possible.

    [*]
    # 4 spaces
    indent_style = space
    indent_size = 4
    # Consistent EOL
    end_of_line = lf
    # Consistent charset
    charset = utf-8
    # Consistent terminators
    trim_trailing_whitespace = true
    insert_final_newline = true
